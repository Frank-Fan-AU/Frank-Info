# 如何为你的Vercel项目设置自定义域名

这篇博客将告诉你如果你从其他提供商购买域名，如何为你的Vercel项目设置自定义域名。

## 步骤1：购买域名

首先，你需要从一个域名提供商那里购买域名。一些流行的选择包括：
- Cloudflare
- Namecheap
- GoDaddy
- Google Domains

## 步骤2：在Vercel中添加域名

1. 登录你的Vercel账户
2. 选择你的项目
3. 转到Settings → Domains
4. 点击"Add Domain"
5. 输入你的域名（例如：example.com）

## 步骤3：配置DNS记录

Vercel会提供DNS记录，你需要在你的域名提供商那里添加这些记录：

### 对于Cloudflare用户：
1. 登录Cloudflare
2. 选择你的域名
3. 转到DNS设置
4. 添加以下记录：
   - 类型：A
   - 名称：@
   - 内容：76.76.19.19
   - 代理状态：DNS only

### 对于其他提供商：
添加类似的A记录，指向Vercel提供的IP地址。

## 步骤4：等待DNS传播

DNS更改可能需要几分钟到48小时才能完全传播。你可以使用在线工具检查DNS传播状态。

## 步骤5：验证设置

一旦DNS传播完成，你的自定义域名应该可以正常访问你的Vercel项目了。

## 常见问题

**Q: 我可以添加子域名吗？**
A: 是的，你可以添加子域名，比如blog.example.com。

**Q: 设置需要多长时间？**
A: DNS传播通常需要几分钟到几小时，具体取决于你的域名提供商。

**Q: 我可以使用多个域名吗？**
A: 是的，Vercel允许你为同一个项目添加多个域名。
